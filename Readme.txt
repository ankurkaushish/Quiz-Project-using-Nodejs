This is the Quiz project based on Nodejs.The Backend Work is done in MongoDB and frontend is in pug , html ,Css.
In this project the Admin will add Question (with Answer) ,The admin can also delete questions , Then the users will answer that question , if the answer is right then Score will be incremented , otherwise the 
Score would be decremented.
